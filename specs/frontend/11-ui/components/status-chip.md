
# Component: StatusChip

## Responsabilidade
Exibir status visual (Draft / Active / Disabled).

## Material base
- M3 Assist/Filter Chip

## Props
| Nome | Tipo | Obrigatório |
|----|----|----|
| status | Draft | Active | Disabled |

## Regras visuais
- Draft → surfaceVariant
- Active → primary
- Disabled → error
